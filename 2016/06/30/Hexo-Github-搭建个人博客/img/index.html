Hexo + Github 搭建个人博客

作者博客: Julius-git.github.io

简介


Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

<一：搭建HEXO>


环境
Node.js(自带npm)
Git

安装
安装 Hexo 只需几分钟时间，若您在安装过程中遇到问题或无法找到解决方式，请留下评论，我会尽力解决您的问题。在这里就不着重将Node.js与Git的安装了，网上有很多教程可以参考。在这里假设你已经安装了Node.js 与 Git
接下来只需要使用 npm 即可完成 Hexo 的安装。

$ npm install -g hexo-cli

Init
安装 Hexo 完成后，请执行下列命令，Hexo 将会在指定文件夹中新建所需要的文件。 指定文件夹路径，默认当前所在文件夹

$ hexo init .
$ cd .
$ npm install

新建完成后，指定文件夹的目录如下：
.
├── _config.yml(后面需要配置此文件)
├── package.json
├── scaffolds
├── scripts
├── source
| ├── _drafts
| └── _posts
└── themes
Demo

$ hexo new “Hello Hexo”

Run

$ hexo s

更多Hexo 基本操作请查阅官方文档: hexo.io

<二：配置GitHub>

如果没github的用户,自行注册一个吧。 还需授权ssh key.

1.New Repository
点击 new repository 新建一个新的库

2.Set Repostiory Name
注意，这里的名字必须为你的用户名为前缀加上github.io作为你的二级域名
如我的用户名Julius-Git 名字设为 Julius-Git.github.io
其它默认，点击Create reposltory

3.Copy HTTPS Link
复制生成的HTTPS链接

4.Seting Hexo Config
在根目录下设置_config.xml 文件，翻到最下面，设置如图所示，将repo 换成你自己的地址 ，保存退出。


5.Generate and Deploy
使用 Generate 生成静态文件,Deploy 一键部署到你的github上. 输入 用户名.github.io 访问你的博客

$hexo g
$hexo d

End
